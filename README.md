# SatNotes App 🪙

**SatNotes** is a physical cryptocurrency bearer instrument platform. This application integrates a live mapping interface with the **SatNotes Digital Transfer Protocol (SDTP)** to bridge the gap between physical satoshi-denominated banknotes and digital Bitcoin liquidity.

## 🛠 Features

- **3-Level Shopping Hub**: A nested hierarchy of goods and services (Fashion, Tech, Travel, etc.).
- **Smart Logic Gate**: Automatically switches between **Map Mode** (for local ASAP needs) and **Escrow Mode** (for online BTC/SDTP purchases).
- **Travel Integration**: Route planning directly on a live Google Maps background.
- **SatNotes Prioritization**: Local map pins are ranked based on SatNotes acceptance and protocol status.

---

## 🔐 The SatNotes Protocol (SDTP)
Unlike traditional paper wallets, SatNotes are physical banknotes that must be **physically destroyed** to trigger the redemption of escrowed Bitcoin vaults. This app serves as the verification interface for these instruments, and the localization of physical exchange, issuance, and redemption points.
